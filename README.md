# Half Frame Film Splitter

하프 프레임 필름 카메라로 촬영한 스캔 이미지를 두 장으로 분리하고, 수동 편집본과 자동 처리본이 섞인 폴더를 정리해주는 웹 앱.

🔗 **[바로 사용하기](https://sternjeong.github.io/side-project/)**

---

## 배경

하프 프레임 카메라는 35mm 필름을 17mm씩 두 칸으로 나눠 촬영한다.
스캔하면 두 장의 사진이 좌우(또는 상하)로 붙어 있고, 그 사이에 약 1mm의 어두운 틈이 생긴다.
이 앱은 그 틈을 자동으로 감지해 사진을 두 파일로 분리한다.

자동 처리 후 일부 프레임을 수동으로 재편집하면, 같은 이름의 파일이 중복 생성된다.
**파일 중복 필터**는 이 중복을 정리해 수동 편집본만 남겨준다.

## 도구 1 — Half Frame Film Splitter

### 주요 기능

- **자동 감지** — 이미지 중앙 영역의 밝기·균일도를 분석해 틈 위치를 자동으로 찾아줌
- **수동 조정** — 슬라이더로 분할 위치(L / R)를 세밀하게 조정 가능
- **좌우 / 상하 분할** — 스캔 방향에 따라 선택 가능
- **배치 처리** — 여러 파일을 한 번에 자동 처리 및 저장
- **화질 보존** — 원본 해상도 그대로 크롭, JPG 97% / PNG 무손실 출력
- **HTML 파일 저장** — 메뉴바에서 앱 자체를 단일 HTML 파일로 다운로드 가능

### 사용법

1. 브라우저에서 [앱](https://sternjeong.github.io/side-project/) 열기
2. 스캔된 하프 프레임 이미지를 드래그하거나 클릭해서 불러오기
3. 자동 감지 결과 확인 후 필요 시 슬라이더로 미세 조정
4. **⚡ 전체 자동 처리 및 저장** 또는 **✂️ 이 파일 처리 및 저장** 클릭
5. 각 프레임이 `원본파일명_frame1.jpg` / `_frame2.jpg` 로 저장됨

### 지원 포맷

| 입력 | 출력 |
|------|------|
| JPG / JPEG | JPG (품질 97%) |
| PNG | PNG (무손실) |
| WebP | WebP (품질 97%) |

## 도구 2 — 파일 중복 필터

자동 분할 후 일부 프레임을 수동으로 재편집하면 아래와 같은 중복 파일이 생긴다.

```
83980037_frame2.JPG        ← 자동 처리본 (삭제 대상)
83980037_frame2 (1).JPG    ← 수동 편집본 (유지)
```

OS가 동일한 이름의 파일을 저장할 때 자동으로 ` (1)` 또는 `_(1)` suffix를 붙인다.
이 필터는 그 패턴을 감지해 원본(자동 처리본)을 제거하고 편집본만 남긴다.

### 사용법

1. 정리할 파일들을 드래그 앤 드롭
2. **✂️ 중복 제거** 클릭 — 원본 파일이 목록에서 제거됨
3. **💾 전체 저장** 클릭 — 남은 파일들을 일괄 다운로드

### 인식하는 패턴

| 수동 편집본 | 삭제되는 원본 |
|---|---|
| `photo (1).JPG` | `photo.JPG` |
| `photo_(1).JPG` | `photo.JPG` |

### CLI 대안 (터미널에서 직접 실행)

폴더로 이동한 뒤 아래 명령어를 실행하면 동일한 동작을 수행한다.

**Windows — PowerShell**
```powershell
Get-ChildItem | ForEach-Object {
    if ($_.Name -match '^(.+)[_ ]\((\d+)\)(\.[^.]+)$') {
        $orig = $Matches[1] + $Matches[3]
        if (Test-Path $orig) { Remove-Item $orig; Write-Host "삭제: $orig" }
    }
}
```

**Mac / Linux — bash**
```bash
for f in *; do
    if [[ "$f" =~ ^(.+)[[:space:]_]\(([0-9]+)\)(\.[^.]+)$ ]]; then
        orig="${BASH_REMATCH[1]}${BASH_REMATCH[3]}"
        [ -f "$orig" ] && rm "$orig" && echo "삭제: $orig"
    fi
done
```
