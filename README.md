# Half Frame Film Splitter

하프 프레임 필름 카메라로 촬영한 스캔 이미지를 두 장으로 분리해주는 웹 앱.

## 배경

하프 프레임 카메라는 35mm 필름을 17mm씩 두 칸으로 나눠 촬영한다.
스캔하면 두 장의 사진이 좌우(또는 상하)로 붙어 있고, 그 사이에 약 1mm의 어두운 틈이 생긴다.
이 앱은 그 틈을 자동으로 감지해 사진을 깔끔하게 두 파일로 분리한다.

## 주요 기능

- **자동 감지** — 이미지 중앙 영역의 밝기를 분석해 틈 위치를 자동으로 찾아줌
- **수동 조정** — 슬라이더로 분할 위치와 틈 제거 폭을 세밀하게 조정 가능
- **좌우 / 상하 분할** — 스캔 방향에 따라 선택 가능
- **화질 보존** — 원본 해상도 그대로 크롭, JPG는 97% 품질 / PNG는 무손실 출력
- **개별 저장** — 각 프레임을 개별 파일로 다운로드

## 사용법

1. `index.html`을 브라우저에서 열기 (별도 서버 불필요)
2. 스캔된 하프 프레임 이미지를 드래그하거나 클릭해서 불러오기
3. 자동 감지 결과 확인 후 필요 시 슬라이더로 미세 조정
4. **분할 실행** 버튼 클릭
5. 각 프레임을 저장

## 지원 포맷

| 입력 | 출력 |
|------|------|
| JPG / JPEG | JPG (품질 97%) |
| PNG | PNG (무손실) |
| WebP | WebP (품질 97%) |

## 기술 스택

- 순수 HTML / CSS / JavaScript (외부 라이브러리 없음)
- Canvas API — 이미지 크롭 및 내보내기
- FileReader API — 로컬 파일 읽기
- UI 테마: Windows XP Luna
